Minestom Tools - Anvil to Polar Converter

Instructions:

  1. Find your world folder.
  2. Create a zip file containing the entire world folder.
  3. Upload the zip file using the button below.
  4. Wait for the conversion process to complete.
  5. Download the converted Polar format world.

In Case of Error:

  1. Open minecraft.
  2. Put the unzipped world folder into your .minecraft/saves folder.
  3. Open single player.
  4. Click on the world and then click edit and "Optimize World"
  5. Redo the instructions.
